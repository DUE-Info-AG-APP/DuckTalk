A simple messaging service.

Message format:
{
  "sender": "user_x",
  "recipient": "user_y",
  "timestamp": "YYYY-MM-DD HH:MM:SS",
  "data": "bla bla bla"
}

Message model:
int id (generated by the server)
String sender
String recipient
Timestamp timestamp
String data

User model:
String username
String password
Array<String> contacts
